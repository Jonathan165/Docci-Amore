🎂 Site da Confeitaria

Projeto acadêmico desenvolvido em equipe com foco na criação de um site one-page, moderno, elegante e responsivo, utilizando apenas HTML5 e CSS3.

O objetivo é apresentar a identidade visual de uma confeitaria, seus produtos, história, galeria de fotos e formas de contato em uma experiência visual agradável e organizada.

📌 Objetivo do Projeto

Criar um site profissional para confeitaria contendo:

Barra de anúncio promocional
Menu de navegação
Página inicial (Hero Section)
Cardápio / Produtos
Sobre a confeitaria
Galeria de imagens
Depoimentos de clientes
Contato / Localização
Rodapé institucional
🛠️ Tecnologias Utilizadas
HTML5
CSS3
Google Fonts
Layout Responsivo
Organização modular por arquivos CSS
